**Generative Extend** is an AI-powered video editing feature that allows editors to **extend the duration of a video clip by generating new frames** that blend naturally with the original footage.

Instead of freezing a frame or slowing down motion, Generative Extend **creates brand-new visual content** that continues the scene smoothly.

1. The AI analyzes the last frames of a clip:

3. New frames are generated using generative models.

4. These frames are appended to the original clip, extending its length seamlessly.
